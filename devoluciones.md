### Idea y diseño

- Diseño simple. Es prolijo y funcional, aunque recuerda a plantillas genéricas de HTML o basadas en componentes de Bootstrap. Cumple con el contenido que presenta.
- Recomendamos evitar la combinacion texto centrado (titulos) con texto alinado a la izquierda (contenido)

### HTML

- Bien en general.
- Buena estructura de títulos, aunque sería conveniente que cada pagina tenga un h1 distinto.
- Uso de elemntos semánticos

### CSS

- Bien, usa muchos de los conceptos vistos en clase como rems, animaciones, etc.
- Bien el uso de variables

### Responsive

- Se aplican tamaños fluidos en general
- Se usan media-queries para adaptaciones puntuales

### Uso de Astro o JS

- Usa Astro y JS
- Recomendamos usar el componente Image de Astro para optimizar las imagenes.
